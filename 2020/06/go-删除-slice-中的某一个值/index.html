Go 删除 Slice 中的某一个值

2020-06-03

文章目录

    • 方法一
    • 方法二
    • Benchmark
    • 参考

【注意】最后更新于 June 3, 2020，文中内容可能已过时，请谨慎使用。

方法一

优点: 速度最快 缺点: 会导致切片数据顺序改变

1
2
3
4
5
6
7
8

a := []string{"A", "B", "C", "D", "E"}
i := 2

a[i] = a[len(a)-1] // 将数组的最后一位赋值给需要删除的 index 上
a = a[:len(a)-1] // 移除掉最后一个没用的数据

// Output:
// [A B E D]

方法二

优点: 速度会随着切片长度改变 缺点: 保持原有切片顺序

1
2
3
4
5
6
7

a := []string{"A", "B", "C", "D", "E"}
i := 2

a = append(a[:i], a[i+1:]...)

// Output:
// [A B D E]
