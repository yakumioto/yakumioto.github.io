Go 删除 Slice 中的某一个值

六月 3, 2020 · 1 分钟 · Mioto Yaku
目录
  • 方法一
  • 方法二
  • Benchmark
  • 参考

方法一#

优点: 速度最快 缺点: 会导致切片数据顺序改变

a := []string{"A", "B", "C", "D", "E"}
i := 2

a[i] = a[len(a)-1] // 将数组的最后一位赋值给需要删除的 index 上
a = a[:len(a)-1] // 移除掉最后一个没用的数据

// Output:
// [A B E D]

方法二#

优点: 速度会随着切片长度改变 缺点: 保持原有切片顺序

a := []string{"A", "B", "C", "D", "E"}
i := 2

a = append(a[:i], a[i+1:]...)

// Output:
// [A B D E]
